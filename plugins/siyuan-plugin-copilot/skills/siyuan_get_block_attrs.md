获取指定块的属性。

## 使用场景
- 读取某个块的自定义属性（如 tags、bookmark、alias 等）

## 参数
- id: 要查询的块ID

## 返回
- 返回一个对象，键为属性名，值为属性值。

## 对文档块返回的所有属性
- icon: 文档图标
- id: 文档 id
- tags: 文档标签，多个标签使用逗号分隔
- type: 文档类型（'d' 表示文档）
- update: 更新时间戳或时间字符串
- bookmark: 书签标识（如存在）
- alias: 文档别名
- name: 文档命名，可与title不同
- title: 文档标题
- memo: 备注

## 普通块不包含的属性
- icon
- tags
- title

## 注意
- 不包含文档路径、归属笔记本等信息，需要通过sql查询
