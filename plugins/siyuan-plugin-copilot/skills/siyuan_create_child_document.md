在指定父文档下创建子文档的工具。

## 何时使用
- 需要在指定文档下创建子文档
- 用户要求在某个文档内部创建新文档
- 整理层级笔记结构

## 文档创建功能
1. 自动在父文档路径下创建子文档
2. 支持Markdown格式创建文档

## 使用示例

```javascript
// 在父文档下创建子文档
siyuan_create_child_document({
  parentId: "20210808180117-6v0mkxr",  // 父文档ID
  title: "子文档标题",                 // 子文档标题（不含路径）
  markdown: "# 子文档内容\n\n这是内容..."
})
```

## 参数说明
- parentId: 父文档ID（必填）
- title: 子文档标题（必填，不含路径，仅文档名）
- markdown: 文档内容，使用Markdown格式（必填）
