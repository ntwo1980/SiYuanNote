在思源笔记中创建新文档的工具。

## 何时使用
- 用户要求创建新笔记
- 需要基于对话内容生成文档
- 整理信息并保存为新文档

## 文档创建功能
1. 自动创建层级目录
2. 支持完整的Markdown格式，Markdown内容会被解析并转换为块

## 属性说明
- `notebook`: 笔记本ID（可选，未指定时保存在默认笔记本中，不要自行选择）
- `path`: 文档保存的目录路径。必须以 `/` 开头，如 `/日记/2024/01`。此路径不应包含最后的文档名称/文件名。
- `title`: 新文档的标题（文件名），如 `量子力学读书笔记`。
- `markdown`: 文档内容，使用Markdown格式。

## 使用示例

```javascript
siyuan_create_document({
  notebook: "20210808180117-6v0mkxr",
  path: "/代码开发和插件开发/@思源笔记插件开发丨任务笔记管理插件",
  title: "思源笔记任务笔记管理插件丨使用文档",
  markdown: "## 关于插件使用\n\n- 这是一个非常强大的任务管理插件……"
})
```

## 注意事项
- 思源笔记会自动将 `title` 参数作为新文档的标题，因此不需要在 Markdown 内容中再使用一级标题（# 标题）作为笔记的标题。
- 如果用户没指定笔记本和保存的路径目录，可以询问用户是存储在默认路径还是指定路径，也可以使用默认目录。
