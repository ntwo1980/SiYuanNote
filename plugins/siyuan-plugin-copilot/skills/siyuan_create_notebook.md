创建新笔记本的工具。

## 何时使用
- 用户要求创建新的笔记本
- 需要为特定项目或主题创建独立的笔记本
- 组织和管理笔记结构

## 创建功能
- 创建指定名称的新笔记本
- 自动生成笔记本ID
- 新笔记本会自动打开

## 使用示例

```javascript
// 创建项目笔记本
siyuan_create_notebook({
  name: "项目管理"
})

// 创建学习笔记本
siyuan_create_notebook({
  name: "机器学习笔记"
})
```

## 注意事项
- 笔记本名称不能为空
- 如果同名笔记本已存在，可能会报错
- 创建成功后会返回笔记本对象，包含ID等信息
- 新笔记本默认会自动打开
