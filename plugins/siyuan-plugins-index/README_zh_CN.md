# 目录插件

插件会在顶栏右侧生成一个插件图标

左键点击插件图标，在当前文档下插入目录列表，右键显示更多操作

`CTRL + ALT + I` 插入目录 

`CTRL + ALT + O` 插入当前文档大纲

🚀**测试功能**：

为目录以及其它列表创建数据库。
加入数据库后，可通过 `alt + click` 单元格来批量配置属性。
